## Project Standards

### Conventions (from AGENTS.md)
- Conventional commits: `feat:`, `fix:`, `refactor:`, `docs:`, `test:`, `chore:`, `ci:`
- Python 3.13+ — modern syntax (type hints, `X | Y` unions, f-strings)
- All public functions must have docstrings
- No `print()` in production — use logging
- Imports: stdlib → third-party → local, blank line between groups
- Max line length: 120 chars
- `pathlib` or `os.path.dirname(__file__)` for file paths
- CSS variables in `style.css` — no inline styles
- HTMX forms POST to `/actions/*` (form-encoded), NOT `/api/*` (JSON)
- In-memory SQLite for tests — never touch production DB
- Coverage target: > 90%

### Architecture
- Single process: Starlette mounts fastmcp at `/mcp`, web at `/`, API at `/api/`
- `TaskboardStore` is single source of truth — all routes/tools delegate to it
- `store.py` owns all SQL — no raw SQL in routes, templates, or MCP tools
- Schema is immutable (production data)
